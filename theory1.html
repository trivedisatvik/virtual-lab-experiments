Theory

Introduction: HTML is abbreviated for Hyper Text Markup Languages which describes the structure of a web page. The elements of HTML tell the browser how to display the content correctly.

Creating an HTML document : We would recommend working with Sublime Text Editor.From a powerful, custom cross-platform UI toolkit, to an unmatched syntax highlighting engine, Sublime Text sets the bar for performance. Download it from :https://www.sublimetext.com/ and install it on your system. Once it's installed you can open a new file and save it as: name.html .html is the extension of the html files.

Browser : The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and display them correctly. A browser does not display the HTML tags, but uses them to determine how to display the document

HTML Elements : An HTML element is defined by a start tag, some content, and an end tag.

<tagname>Content goes here...</tagname>

The HTML element is everything from the start tag to the end tag.

<h1>My First Heading</h1>

<p>My first paragraph.</p>

The <!DOCTYPE html> declaration defines that this document is an HTML5 document

The <html> element is the root element of an HTML page

The <head> element contains meta information about the HTML page

The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)

The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

The <h1>,<h2>,<h3>,<h4>,<h5>,<h6> elements define headings with decreasing size from h1 being the largest and h6 being the smallest.

The <p> element defines a paragraph

The <br> tag is a self closing tag i.e. it doesn't require a closing tag and is used for creating a line break.

The <hr> tag similar to the <br> tag is a self closing tag and is used to create a horizontal line. Its default color is white.

The <b> element boldens the text.

The <i> element italicize the text.

The <u> element is used for creating an underline below the text.

The <ul> element creates an unordered list with bulleted points. It contains <li> elements for inserting list items one by one.

The <ol> element creates an ordered list with numbers. Like <ul>, <ol> element also contains <li> elements within it.
